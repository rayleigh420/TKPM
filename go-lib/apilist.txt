###
Sign in
POST http://localhost:8080/login
Content-Type: application/json

{
    "email":"user2@gmail.com",
    "password":"123"
}

Result:
Success:
{
  "status": "accepted"
}
Not:
{
  "error": "Password incorrect"
}
Not:
{
  "error": "email not found"
}
